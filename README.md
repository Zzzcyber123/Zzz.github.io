## 为什么推荐付费机场？

免费的 VPN 往往存在以下问题：

- 速度慢
- 高峰期严重拥堵
- 节点数量少
- 广告多
- 数据安全无法保证
- 随时可能停止服务

付费机场通常拥有更多服务器、更大的带宽以及更及时的维护，因此稳定性通常更高。注意: 请不要在任何机场订阅长期套餐，防止机场跑路(停止服务不退款)

## 如果你是翻墙小白，那么你得先了解一下VPN的工作原理。

直连（VPN未连接）时：大概为：你的设备-路由器-GFW(中国网络防火墙)-DNS解析服务器-原路返回。 比如你要直连访问Google，请求在经过GFW时被检测到在黑名单范围内，随即你的请求就被GFW执行DNS 污染、TCP 重置（RST）、SNI/HTTP Host 检测、IP 封锁、主动探测、流量特征识别 等，具体方式会随时间和协议变化，然后成功阻止了你的访问

使用中转VPN时：你的设备-路由器，然后伪装成一个加密的正常流量经过GFW，最后由代理服务器转发至目标网站。由于流量经过加密，GFW 通常无法直接识别访问内容，因此代理协议能够正常建立连接，但是如果你连接的VPN服务器有大量的人进行了过于明显的VPN流量传输，那么GFW有可能会因为传输特征明显是VPN而阻止请求，然后原本正常工作的节点突然连接超时，还有比如最近GFW严格打击翻墙用户也会导致机场节点偶发不稳定问题。

使用专线VPN时：跨境流量最终仍然会经过运营商的国际出口，部分商业专线或中转架构与普通公网代理的实现方式不同，实际体验和抗干扰能力通常更好，因此通常具有较高的可用性，而且一般延迟更低，缺点是更贵

有的地区已经开始实现“墙中墙”，也就是在GFW标准限制的基础上限制更多服务的访问，所以机场几乎需要不停维护服务器和更新技术来尽量保证可用性和稳定性

## 接下来介绍代理软件中的代理模式，规则(Shadowrocket上叫配置)、全局(Shadowrocket上叫代理)

规则: 代理软件为你做好了什么软件该走代理(翻墙)，什么软件该直连，例如你开着规则，在你使用微信时，默认直连，不走代理，不然流量需要走到代理服务器然后又走回来，例如你连着美国节点，那流量就要绕大半个地球，延迟很高。海外软件/网站，像YouTube/X就会自动走代理。注意: 规则模式无法覆盖所有应用场景，需要时请切换至全局，例如Steam用Shadowrocket的配置模式(规则)就无法正常加载，必须使用全局(代理)

全局: 所有流量都经过代理
